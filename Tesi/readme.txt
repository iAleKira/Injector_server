Nif.jar - executable that launches the server framework developed.
Framework.sh - script that automates the whole project.
Filter - Eclipse project containing the All_payloads_filter.jar executable source code.
Payloads - directory containing some payloads files.
PayloadServer - Maven project containing the Nif source code.
TextFiles - directory containing text files that will be used by the server and filter such as nmap-service-probes.txt.
DockerFile - dockerfile to mount docker image
